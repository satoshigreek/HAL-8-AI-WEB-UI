Connect Ollama, vLLM, OpenRouter or any OpenAI-compatible endpoint — including HAL 8 metered inference — and switch between them mid-conversation.

Deploy

On your server in three commands

# point chat.hal8.ai at your server, then: $ git clone https://github.com/satoshigreek/HAL-8-AI-WEB-UI.git $ cd HAL-8-AI-WEB-UI && printf 'WEBUI_SECRET_KEY=%s\nHAL8_DOMAIN=chat.hal8.ai\n' "$(openssl rand -hex 32)" > .env $ docker compose -f docker-compose.prod.yaml up -d --build # HTTPS is automatic. First account created becomes admin.

The production stack ships in the repo: the HAL 8 app behind a Caddy reverse proxy with automatic TLS certificates. All persistent data lives in a single Docker volume.

Full walkthrough — DNS, updates, backups and operating notes — in DEPLOYMENT.md.
